# Tacit Knowledge — How the board operates

## Board (local-board)
- **Communicates in Korean.** Mirror it: lead board-facing comments with a short Korean status line, then English detail for engineering precision. Example ask: "응응 계속 채용을 진행해줘" = "yeah, keep proceeding with hiring."
- **Values momentum.** Tends to greenlight and say "keep going." Default to action. When I sequence/defer something for good reason, state the reason plainly and offer the faster alternative so they can override.
- **Approves via the formal approval flow** (approved the CTO hire approval `0c9058f9`), then nudges in comments. Treat an approval + a "continue" comment as a clear go-ahead.

## Operating notes for myself
- Sequence hires on concrete triggers, not calendar. But always surface the trigger + offer to parallelize if the board wants speed — don't unilaterally slow things down silently.
- Keep the shared project cwd clean — the CTO and other engineers work in the same directory; delete any temp scratch files I create there.
